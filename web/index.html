WebChuGL

ChuGL ⇒ now on the Web

Real-time graphics and audio programming in the browser with ChuGL (ChucK Graphics Library) compiled to WebAssembly, WebGPU rendering, and Web Audio! Create and share interactive audiovisual experiences, audio-centric games, data sonifications, and more — all in your web browser!

View on GitHub Documentation Changelog

WebChuGL brings ChuGL, the real-time graphics framework for the ChucK programming language, to the web browser! ChuGL’s C++ source code has been compiled with Emscripten to WebAssembly (WASM) to render via the WebGPU API and run audio through an AudioWorkletNode.

WebChuGL builds on WebChucK by adding ChuGL’s scenegraph, materials, textures, shaders, post-processing, and the full GGen (graphics generator) ecosystem — all tightly synchronized to ChucK’s strongly-timed audio engine.

Features

🎨

WebGPU Rendering

Hardware-accelerated graphics via the modern WebGPU API! Scenegraph, materials, textures, custom shaders, post-processing effects, and more.

🎵

Strongly-Timed Audio

ChucK’s audio engine runs via AudioWorklet with lock-free ring buffers for sample-accurate timing, synthesis, effects, and audio.

🔌

ChuGin Support

Web-compatible ChuGins (ChucK plugins) can be imported and used in WebChuGL! A handful of pre-compiled ChuGins are available here.

🎮

Input & Sensors

Keyboard, mouse, gamepad, MIDI, accelerometer, gyroscope, drag-and-drop files, speech recognition, MediaPipe hand tracking, and more!

🌐

JS ↔ ChucK Bridge

Full bidirectional communication between JavaScript and ChucK! Set and get globals, listen for events, load audio dynamically, and more.

⚙️

Configurable Audio

Set sample rate, channel count, and more via URL params (?srate=44100&out=4) or the init config.

Getting Started

All you need is a browser with WebGPU support (Chrome 113+, Edge 113+, Firefox Nightly). Create an index.html:

<canvas id="canvas"></canvas>
<script type="module">
    import ChuGL from 'https://cdn.jsdelivr.net/npm/webchugl/+esm';

    var ck = await ChuGL.init({
        canvas: document.getElementById('canvas'),
    });

    ck.runCode('SinOsc s => dac; while(true) GG.nextFrame() => now;');
</script>

See the full getting started guide for self-hosted builds, deployment, cross-origin isolation, and building from source.
