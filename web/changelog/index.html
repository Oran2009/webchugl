WebChuGL release history

Releases

v0.1.6

2026-02-23

Fixed

  • Letterbox no longer sets inline styles on the host’s parent element and now resets on VM clear

v0.1.5

2026-02-23

Added

  • TypeScript types (ChucK, AudioConfig, ShredInfo, etc.) are now included in the npm package

Changed

  • ChuGL no longer overwrites document.title with the FPS counter on web builds. The page's <title> is now preserved by default. Use GWindow.title() to set a custom title from ChucK code.

Fixed

  • printErr now uses _printCallback, falling back to console.error

v0.1.4

2026-02-23

Changed

  • Entire audio pipeline now uses planar buffers, eliminating all conversions to the interleaved format.
  • Ring buffer read/write operations use memcpy (C++) and Float32Array.set() (JS) instead of per-sample loops

Fixed

  • Canvas sizing, resizing, and letterboxing now use parent container dimensions
  • Audio initialization race condition
  • Hardcoded <canvas> selector in GLFW
  • GLFW swallowing keyboard events when canvas is not focused

v0.1.3

2026-02-22

Changed

  • Rewrite runtime in TypeScript (added esbuild dev dependency)

Fixed

  • Canvas element no longer hardcoded to #canvas
  • ChuMP package loading now resolves the latest version automatically when no version is specified
  • ChuMP package downloads use a CORS proxy fallback for cross-origin servers

v0.1.2

2026-02-21

Added

  • Website is live!

Changed

  • Runtime is now hosted on the CDN

v0.1.1

2026-02-19

Added

  • Published to npm as an ESM package (webchugl)
  • MIT License

v0.1.0

2026-02-18

Added

  • Initial release — ChucK + ChuGL running in the browser via WebAssembly
  • WebGPU rendering with ChuGL graphics API
  • Web Audio playback via lock-free ring buffers and AudioWorkletProcessor
  • Multi-channel audio input/output support
  • Dynamic ChuGin loading (dlopen side modules)
  • Service worker for COOP/COEP header injection and offline caching
  • Virtual filesystem with bundle.zip extraction and ChucK @import support
  • Gamepad, MIDI, touch, and gyroscope/accelerometer input
  • Project website with interactive examples
